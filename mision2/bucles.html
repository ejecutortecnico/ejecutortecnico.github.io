Bucles en JavaScript

Los bucles nos permiten ejecutar un bloque de código varias veces según una condición. Exploraremos los tipos de bucles más comunes en JavaScript.

Bucle for

El bucle for se utiliza cuando sabemos cuántas veces queremos que se ejecute el código.

// Ejemplo de bucle for for (let i = 0; i < 5; i++) { console.log('Iteración número: ' + i); }

Bucle while

El bucle while se ejecuta mientras una condición sea verdadera.

// Ejemplo de bucle while let contador = 0; while (contador < 5) { console.log('Contador: ' + contador); contador++; }

Bucle do...while

El bucle do...while siempre ejecuta el bloque de código al menos una vez antes de verificar la condición.

// Ejemplo de bucle do...while let contador = 0; do { console.log('Contador: ' + contador); contador++; } while (contador < 5);

Bucle for...of

El bucle for...of se usa para recorrer iterables, como arrays.

// Ejemplo de bucle for...of let numeros = [10, 20, 30]; for (let numero of numeros) { console.log('Número: ' + numero); }

Conclusión

Los bucles son esenciales para la programación, facilitando la repetición de tareas de manera eficiente.
