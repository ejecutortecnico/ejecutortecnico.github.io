Introducción a Python

Conceptos Básicos

Aprende los fundamentos de Python, un lenguaje popular para desarrollo web, ciencia de datos, IA y más.

1. ¿Qué es Python?

Python es un lenguaje de programación de alto nivel, fácil de aprender y con una sintaxis clara y legible.

  • Interpretado
  • Multiparadigma: Soporta programación orientada a objetos y funcional
  • Popular en ciencia de datos, inteligencia artificial, y desarrollo web

2. Sintaxis Básica

Ejemplo de un programa "Hola, mundo" en Python:

print("Hola, mundo")

3. Variables

En Python, no necesitas declarar el tipo de las variables.

nombre = "Juan"
edad = 25
pi = 3.14159

4. Tipos de Datos

Algunos de los tipos de datos comunes en Python:

  • int: Números enteros
  • float: Números decimales
  • str: Cadenas de texto
  • bool: Valores booleanos (True/False)
  • list: Lista de elementos

5. Operadores

Python soporta varios operadores:

  • Aritméticos: +, -, *, /, %
  • Comparación: ==, !=, <, >, <=, >=
  • Lógicos: and, or, not

6. Estructuras de Control

Python usa indentación para definir bloques de código:

# Condicional
if edad >= 18:
    print("Es mayor de edad")
else:
    print("Es menor de edad")

# Bucle
for i in range(5):
    print(i)

7. Funciones

Las funciones en Python permiten organizar y reutilizar el código:

def saludar(nombre):
    print("Hola,", nombre)

saludar("Ana")

Conclusión

Python es un lenguaje versátil y fácil de aprender. Dominar sus fundamentos te permitirá realizar proyectos poderosos y variados.
