Python Intermedio

Temas Avanzados

Exploraremos estructuras de datos y técnicas de manejo de errores en Python.

1. Listas en Python

Las listas son colecciones ordenadas y mutables de elementos.

# Crear una lista
mi_lista = [1, 2, 3, 4]

# Acceder a elementos
print(mi_lista[0])  # Output: 1

# Añadir elementos
mi_lista.append(5)

2. Diccionarios en Python

Los diccionarios almacenan pares clave-valor.

# Crear un diccionario
mi_diccionario = {"nombre": "Juan", "edad": 30}

# Acceder a valores
print(mi_diccionario["nombre"])  # Output: Juan

# Añadir un nuevo par clave-valor
mi_diccionario["ciudad"] = "Madrid"

3. Manejo de Errores

Usamos try-except para capturar errores y evitar que el programa se detenga inesperadamente.

try:
    resultado = 10 / 0
except ZeroDivisionError:
    print("No se puede dividir por cero")

4. Módulos en Python

Los módulos permiten organizar y reutilizar código.

# Importar el módulo math
import math

# Usar funciones del módulo
print(math.sqrt(16))  # Output: 4.0

Conclusión

Con estas herramientas, puedes empezar a construir programas más complejos y organizados en Python.
