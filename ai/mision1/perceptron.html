Ejemplo de Perceptrón

Imagina un perceptrón en tu cerebro que trata de decidir si debes ir a un concierto.

¿El artista es bueno? ¿El clima es favorable? ¿Qué importancia tienen estos factores?

Criterios y Pesos

Criterio Entrada (x) Peso (w)
El artista es bueno x1 = 0 o 1 w1 = 0.7
El clima es bueno x2 = 0 o 1 w2 = 0.6
Un amigo irá x3 = 0 o 1 w3 = 0.5
Se servirá comida x4 = 0 o 1 w4 = 0.3
Se servirá alcohol x5 = 0 o 1 w5 = 0.4

El Algoritmo del Perceptrón

  1. Establecer un valor umbral.
  2. Multiplicar todas las entradas por sus pesos.
  3. Sumar todos los resultados.
  4. Activar la salida.

1. Establecer un Umbral

Umbral = 1.5

2. Multiplicar Entradas por sus Pesos

  • x1 * w1 = 1 * 0.7 = 0.7
  • x2 * w2 = 0 * 0.6 = 0
  • x3 * w3 = 1 * 0.5 = 0.5
  • x4 * w4 = 0 * 0.3 = 0
  • x5 * w5 = 1 * 0.4 = 0.4

3. Sumar los Resultados

Suma ponderada: 0.7 + 0 + 0.5 + 0 + 0.4 = 1.6

4. Activar la Salida

Si la suma es mayor que el umbral, la decisión es positiva.

1.6 > 1.5 → Sí, iré al concierto.
