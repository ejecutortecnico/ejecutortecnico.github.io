Ejemplo: Análisis de Datos de Ventas

  • Conjunto de datos: Ventas de una tienda minorista.
  • Objetivos: Identificar productos más vendidos, analizar tendencias, evaluar impacto de promociones.
  • Técnicas: Limpieza, normalización, visualización con gráficos.
# Limpieza de datos en Python
import pandas as pd
df = pd.read_csv('ventas.csv')
df.fillna(df.mean(), inplace=True)
df.drop_duplicates(inplace=True)
