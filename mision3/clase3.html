Python Avanzado

Temas Complejos

Profundizaremos en técnicas avanzadas para mejorar la eficiencia y organización del código en Python.

1. Funciones Lambda

Las funciones lambda son funciones anónimas que se definen en una sola línea.

# Ejemplo de función lambda
multiplicar = lambda x, y: x * y
print(multiplicar(3, 4))  # Output: 12

2. Comprensión de Listas

Una forma compacta de construir listas en una sola línea.

# Ejemplo de comprensión de listas
cuadrados = [x**2 for x in range(10)]
print(cuadrados)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

3. Manejo de Archivos

Permite leer y escribir en archivos.

# Abrir y leer un archivo
with open('archivo.txt', 'r') as file:
    contenido = file.read()
    print(contenido)

4. Clases y Objetos (POO)

La programación orientada a objetos permite organizar el código en clases y objetos.

# Definición de una clase
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
        print(f"Hola, soy {self.nombre} y tengo {self.edad} años")

# Crear un objeto
persona = Persona("Ana", 30)
persona.saludar()  # Output: Hola, soy Ana y tengo 30 años

Conclusión

Con estos conceptos avanzados, puedes construir aplicaciones y scripts más robustos en Python.
