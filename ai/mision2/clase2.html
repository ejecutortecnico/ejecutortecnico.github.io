¿Qué es una API?

Una API (Interfaz de Programación de Aplicaciones) permite la comunicación entre diferentes sistemas.

Ejemplo de uso de APIs

Las APIs permiten que aplicaciones web utilicen servicios de terceros, como pagos o mapas.

APIs en Python

Python cuenta con librerías como requests para consumir APIs de manera sencilla.

Ejemplo en Python:

import requests
response = requests.get('https://jsonplaceholder.typicode.com/posts')
print(response.json())

Usos de APIs en Inteligencia Artificial

Las APIs de AI permiten acceso a modelos avanzados sin necesidad de entrenarlos desde cero.

Ejemplo: OpenAI, Google Cloud AI, IBM Watson.

Ejemplo de API de AI en Python

import openai
response = openai.ChatCompletion.create(
    model='gpt-4', 
    messages=[{"role": "user", "content": "Hola"}]
)
print(response)

Conclusión

Las APIs facilitan la integración de inteligencia artificial en proyectos.
