1. ¿Qué es un prompt?

Un prompt es una instrucción que le das a una inteligencia artificial (como ChatGPT) para que genere un resultado. En este caso, queremos que genere código HTML útil y funcional.

2. Estructura básica de un prompt

Un buen prompt debe ser claro, específico y tener contexto. Ejemplo:

Genera una página HTML con un formulario de contacto que tenga campos para nombre, correo y mensaje.

3. Especifica el propósito del código

Di para qué lo necesitas. Esto ayuda a que el resultado sea más útil.

Necesito una página HTML para registrar usuarios a un evento en línea, con nombre, email y botón de enviar.

4. Usa palabras clave técnicas

Menciona etiquetas específicas de HTML si las conoces:

Quiero una tabla HTML con 3 filas y 2 columnas, usando la etiqueta <table>.

5. Pide ejemplos con estilo CSS si es necesario

Si necesitas diseño, dilo explícitamente:

Genera una página HTML con fondo azul y texto blanco usando CSS.

6. Añade restricciones

Esto mejora la precisión. Por ejemplo:

No uses JavaScript, solo HTML y CSS.

7. Define el contexto o público

Ayuda a que el resultado se adapte mejor a quien lo va a usar.

Crea una interfaz HTML sencilla para estudiantes que usan celulares.

8. Compara buenos vs malos prompts

Mal prompt: "Haz una página web."
Buen prompt: "Crea una página HTML con un menú de navegación arriba y tres secciones con imágenes."

9. Ajusta y refina el prompt

Si el resultado no es lo que querías, ajusta el prompt. Es un proceso iterativo.

10. Resume lo aprendido

  • Sé claro y específico
  • Incluye contexto
  • Usa ejemplos y palabras clave
  • Itera y mejora tus prompts
