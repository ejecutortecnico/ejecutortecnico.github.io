¿Qué es una variable en JavaScript?

Una variable en JavaScript es un contenedor para almacenar datos. Se puede usar para guardar números, textos, objetos y más.

Declaración de variables

En JavaScript, puedes declarar una variable usando las palabras clave var, let o const.

// Declarar una variable con let
let mensaje = "Hola, Mundo!";

Asignar un nuevo valor

Una variable declarada con let puede cambiar su valor más adelante:

// Reasignar un nuevo valor a la variable
mensaje = "JavaScript es genial!";

Uso de const

Las variables declaradas con const no pueden cambiar su valor después de ser asignadas.

// Declarar una variable constante con const
const pi = 3.1416;

Operadores y Expresiones

.

Aritméticos

  • `+`, `-`, `*`, `/`, `%`
  • let suma = 10 + 5; // 15
  • let producto = 4 * 3; // 12

Operadores y Expresiones

.

Asignación

  • `=`, `+=`, `-=`, `*=`, `/=`, `%=`
  • let x = 10;
  • x += 5; // x = 15
  • x *= 2; // x = 30

Operadores y Expresiones

.

Comparación

  • `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`
  • let a = 5;
  • let b = "5";
  • console.log(a == b); // true
  • console.log(a === b); // false

Operadores y Expresiones

.

Lógicos

  • `&&`, `||`, `!`
  • let verdadero = true;
  • let falso = false;
  • console.log(verdadero && falso); // false
  • console.log(verdadero || falso); // true
  • console.log(!verdadero); // false.
