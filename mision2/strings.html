Strings en JavaScript

Aprende cómo trabajar y manipular cadenas de texto en JavaScript.

1. Creación de Strings

Los strings pueden ser definidos de varias maneras:

// Comillas simples
let saludo = 'Hola, mundo';

// Comillas dobles
let frase = "JavaScript es asombroso";

// Plantillas literales
let nombre = 'Carlos';
let mensaje = `Hola, ${nombre}`;

2. Propiedad length

Obtén la longitud de un string:

let texto = "JavaScript";
console.log(texto.length); // 10

3. Métodos Comunes de Strings

Algunos métodos útiles:

// Mayúsculas y minúsculas
let texto = "javascript";
console.log(texto.toUpperCase()); // "JAVASCRIPT"
console.log(texto.toLowerCase()); // "javascript"

// Reemplazo de texto
let saludo = "Hola, mundo";
let nuevoSaludo = saludo.replace("mundo", "amigo");
console.log(nuevoSaludo); // "Hola, amigo"

// Eliminación de espacios
let fraseConEspacios = "  Hola, JavaScript   ";
console.log(fraseConEspacios.trim()); // "Hola, JavaScript"

4. Métodos Avanzados de Strings

Otros métodos útiles:

// Buscar una subcadena
let frase = "Bienvenido a JavaScript";
console.log(frase.includes("JavaScript")); // true

// Obtener una parte del string
let subcadena = frase.slice(12, 22);
console.log(subcadena); // "JavaScript"

// Dividir un string en un array
let lista = "manzana,banana,uva";
let arrayFrutas = lista.split(",");
console.log(arrayFrutas); // ["manzana", "banana", "uva"]
