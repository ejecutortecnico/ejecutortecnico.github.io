Variables en JavaScript

Las variables en JavaScript son contenedores para almacenar datos. Se pueden declarar usando var, let y const.

Declaración de Variables

Las variables en JavaScript se pueden declarar con las palabras clave var, let y const.

// Usando var
var nombre = "Ana";

// Usando let
let edad = 25;

// Usando const
const PI = 3.1416;

Var vs Let vs Const

Cada forma de declarar variables tiene sus propias características:

  • var: Tiene un ámbito de función y permite redeclaraciones.
  • let: Tiene un ámbito de bloque y no permite redeclaraciones en el mismo ámbito.
  • const: Se utiliza para variables cuyo valor no cambia. No se puede reasignar ni redeclarar.

Tipos de Datos en JavaScript

JavaScript soporta varios tipos de datos, incluyendo:

  • Números: Representa números, tanto enteros como flotantes. Ejemplo: let numero = 42;
  • Cadenas: Secuencias de texto. Ejemplo: let texto = "Hola, mundo";
  • Booleanos: Valores de verdadero o falso. Ejemplo: let esVerdad = true;
  • Arreglos: Listas de valores. Ejemplo: let numeros = [1, 2, 3, 4];
  • Objetos: Colecciones de propiedades. Ejemplo: let persona = {nombre: "Ana", edad: 25};

Ámbito de Variables

El ámbito de una variable define dónde puede ser utilizada. En JavaScript, las variables pueden tener:

  • Ámbito Global: Las variables declaradas fuera de cualquier bloque tienen un alcance global.
  • Ámbito de Bloque: Las variables declaradas dentro de un bloque (usando let o const) solo son accesibles dentro de ese bloque.
  • Ámbito de Función: Las variables declaradas dentro de una función solo son accesibles dentro de esa función.

Conclusión

Las variables son fundamentales para almacenar y manipular datos en JavaScript. Comprender la diferencia entre var, let y const te ayudará a escribir código más seguro y eficiente.
