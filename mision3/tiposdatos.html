Tipos de Datos en Python

Exploraremos los diferentes tipos de datos que ofrece Python.

1. Números

Python soporta diferentes tipos de números:

entero = 10
flotante = 10.5
complejo = 3 + 4j

2. Cadenas de Texto

Las cadenas se definen usando comillas simples o dobles:

texto1 = "Hola, mundo"
texto2 = 'Python es genial'

3. Listas

Las listas son colecciones ordenadas:

lista = [1, 2, 3, 4, 5]
lista.append(6)
lista.remove(2)

4. Tuplas

Las tuplas son similares a las listas, pero son inmutables:

tupla = (1, 2, 3)
print(tupla[0]) # Accede al primer elemento

5. Diccionarios

Los diccionarios almacenan pares clave-valor:

diccionario = {"nombre": "Juan", "edad": 30}
diccionario["profesión"] = "Ingeniero"

Conclusiones

Python ofrece una variedad de tipos de datos que permiten almacenar y manipular información de diversas maneras. Comprender estos tipos es esencial para programar de manera efectiva en Python.
