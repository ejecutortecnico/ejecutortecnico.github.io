¡Aprendamos JavaScript!

JavaScript es un lenguaje de programación que nos permite hacer cosas divertidas con las páginas web. ¡Vamos a descubrirlo!

¿Qué es JavaScript?

JavaScript es un lenguaje que permite agregar interacción a las páginas web, como hacer que los botones funcionen o que las imágenes cambien.

// Este es un ejemplo de JavaScript
alert("¡Hola, Mundo!");

Variables

Las variables son como cajitas donde guardamos información.

// Un ejemplo de variable
let nombre = "Ana";
alert(nombre);

Funciones

Las funciones son como instrucciones que le damos a la computadora para hacer algo.

// Un ejemplo de función
function saludar() {
    alert("¡Hola a todos!");
}
saludar();

Eventos

Los eventos son acciones que ocurren en una página web, como cuando hacemos clic en un botón.
