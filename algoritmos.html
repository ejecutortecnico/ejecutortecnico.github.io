¿Qué son algoritmos?

Un algoritmo es una serie de pasos o instrucciones organizadas de manera lógica que permiten resolver un problema o realizar una tarea específica.

Características de los algoritmos:

  • Definidos: Cada paso debe ser claro y sin ambigüedades.
  • Finitos: Deben tener un número limitado de pasos.
  • Precisos: Deben producir el mismo resultado bajo las mismas condiciones.

Ejemplo sencillo:

Algoritmo para hacer un sándwich:

  1. Tomar dos rebanadas de pan.
  2. Untar mantequilla o mayonesa.
  3. Colocar los ingredientes deseados.
  4. Cubrir con la otra rebanada de pan.

¿Qué es programar?

Programar es el proceso de escribir instrucciones que una computadora puede entender para realizar una tarea específica.

Propósito de la programación:

  • Automatizar: Realizar tareas repetitivas de manera eficiente.
  • Resolver problemas: Proporcionar soluciones a desafíos reales.
  • Crear aplicaciones: Desde juegos hasta herramientas de negocios.

Lenguajes de programación:

Existen diferentes lenguajes como Python, Java, C++, entre otros, que sirven como herramientas para comunicarnos con las computadoras.

Ejemplo:

Un programa sencillo en Python para sumar dos números:
