Flask Básico

Aprenderemos sobre el framework Flask para crear aplicaciones web.

1. Instalación de Flask

Para instalar Flask, utiliza pip:

pip install Flask

2. Crear una Aplicación Básica

Ejemplo de una aplicación Flask simple:

from flask import Flask
app = Flask(__name__)
@app.route('/')
   def home():
      return "¡Hola, Flask!"
app.run(debug=True)

3. Rutas y Vistas

Definición de varias rutas:

@app.route('/sobre')
   def sobre():
      return "Página sobre"

4. Plantillas con Jinja2

Uso de plantillas en Flask:

from flask import render_template
@app.route('/pagina')
   def pagina():
      return render_template('pagina.html')
