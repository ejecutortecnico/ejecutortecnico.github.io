Introducción al DOM en JavaScript

El DOM (Document Object Model) es la representación de la estructura de un documento HTML o XML. Nos permite acceder y manipular el contenido y estructura de las páginas web mediante JavaScript.

¿Qué es el DOM?

El DOM es una interfaz que representa el documento HTML como una estructura jerárquica de nodos, donde cada elemento HTML es un nodo del árbol DOM.

  • Los nodos pueden ser elementos, atributos o texto.
  • El DOM permite a JavaScript manipular los elementos HTML dinámicamente.
  • Ejemplo de estructura jerárquica del DOM:
<html>
    <body>
        <h1>Título</h1>
        <p>Parrafo</p>
    </body>
</html>

Accediendo a Elementos del DOM

Podemos acceder a los elementos del DOM usando métodos como getElementById, getElementsByClassName o querySelector. Ejemplo:

document.getElementById("miElemento");
document.getElementsByClassName("miClase");
document.querySelector(".miClase");

Modificar Contenido del DOM

Una vez accedemos a los elementos del DOM, podemos modificar su contenido o atributos usando propiedades como innerHTML o textContent. Ejemplo:

document.getElementById("miElemento").innerHTML = "Nuevo contenido";
document.querySelector("h1").textContent = "Nuevo título";

Crear y Eliminar Elementos del DOM

Podemos crear nuevos elementos y agregarlos al DOM usando createElement y appendChild. Para eliminar, usamos removeChild. Ejemplo:

// Crear un nuevo elemento
let nuevoElemento = document.createElement("p");
nuevoElemento.textContent = "Soy un nuevo párrafo";
// Añadirlo al body
document.body.appendChild(nuevoElemento);

// Eliminar un elemento
let elementoAEliminar = document.getElementById("miElemento");
document.body.removeChild(elementoAEliminar);

Conclusión

El DOM es una herramienta poderosa que permite a JavaScript interactuar con las páginas web. Con los métodos y propiedades correctas, puedes crear dinámicamente contenido, modificar páginas y mejorar la interactividad.
