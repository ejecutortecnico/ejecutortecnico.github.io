Curso de Programación Intermedia

Objetivos Generales

  • Desarrollar aplicaciones frontend modernas con React.
  • Dominar el diseño y gestión de bases de datos relacionales con SQL.
  • Construir aplicaciones backend con Python y Flask.
  • Integrar frontend, backend y bases de datos en proyectos completos.
  • Fortalecer habilidades de trabajo en equipo, documentación técnica y buenas prácticas de desarrollo.
  • Impulsar la empleabilidad de los estudiantes hacia roles en tecnología.

Ruta de Empleabilidad

Perfiles y oportunidades laborales

  • Desarrollador Frontend (React, HTML, CSS, JS).
  • Desarrollador Backend (Python + Flask, SQL, APIs REST).
  • Fullstack Developer Junior (React + Flask + SQL).
  • Soporte y mantenimiento de bases de datos.
  • Analista de datos inicial (consultas SQL, reportes).

✅ Al finalizar, los estudiantes estarán listos para roles junior en desarrollo de software, con la capacidad de crecer hacia fullstack developer o especializarse en frontend, backend o bases de datos.

Misión 1

Desarrollo de Frontend con React

En esta misión aprenderás a construir interfaces modernas con React, entendiendo el uso de componentes, props, estado y hooks para desarrollar aplicaciones dinámicas y modulares.

Objetivos de la Misión 1

  • Dominar los fundamentos de React.
  • Crear componentes funcionales y de clase.
  • Manejar estado, props y eventos.
  • Integrar hooks y manejar datos dinámicos.
  • Construir una aplicación React completa en equipo.

Temas de la Misión 1

  • ✅ Introducción a React y JSX.
  • ✅ Componentes, props y estado.
  • ✅ Hooks: useState, useEffect y otros.
  • ✅ Manejo de eventos.
  • ✅ Listas, condicionales y renderizado dinámico.
  • ✅ Integración de conceptos en una app React.
  • ✅ Pruebas básicas y despliegue en Vercel/Netlify.
  • ✅ Cápsulas: buenas prácticas de frontend.
  • ✅ Co–creación: planificación y desarrollo de un proyecto.
  • ✅ Presentación de proyectos finales.

Misión 2

Manejo de Bases de Datos Relacionales con SQL

En esta misión aprenderás a diseñar, crear y manipular bases de datos relacionales con SQL, entendiendo cómo organizar información y garantizar la integridad de los datos para aplicaciones modernas.

Objetivos de la Misión 2

  • Comprender el modelo relacional y sus componentes.
  • Dominar consultas básicas y avanzadas en SQL.
  • Modelar datos con diagramas entidad–relación.
  • Aplicar integridad referencial y normalización.
  • Implementar procedimientos, funciones y triggers.
  • Gestionar usuarios y seguridad en bases de datos.
  • Construir el esquema de base de datos del proyecto final.

Temas de la Misión 2

  • ✅ Introducción a las bases de datos relacionales.
  • ✅ Creación y manipulación de bases de datos (DDL, DML).
  • ✅ Consultas simples y avanzadas (JOINs, subconsultas, agregaciones).
  • ✅ Modelado de datos y diagramas ER.
  • ✅ Integridad referencial y normalización.
  • ✅ Procedimientos almacenados, funciones y triggers.
  • ✅ Seguridad y gestión de usuarios en SQL.
  • ✅ Cápsulas: buenas prácticas en SQL.
  • ✅ Co–creación: diseño e implementación del esquema de BD.
  • ✅ Documentación técnica y consultas en inglés.

Misión 3

Desarrollo Backend con Python y Flask

En esta misión aprenderás a construir APIs y aplicaciones backend con Python y Flask, conectando la lógica del servidor con bases de datos y asegurando autenticación y seguridad.

Objetivos de la Misión 3

  • Dominar los fundamentos de Python aplicados al backend.
  • Construir un servidor con Flask y definir rutas.
  • Desarrollar APIs RESTful conectadas a una base de datos.
  • Manejar autenticación y autorización.
  • Integrar backend con frontend en React.
  • Aplicar buenas prácticas de seguridad en aplicaciones web.
  • Trabajar en co–creación en un proyecto final integrador.

Temas de la Misión 3

  • ✅ Introducción a Python y estructuras de datos.
  • ✅ Fundamentos de Flask y creación de rutas.
  • ✅ Desarrollo de APIs RESTful.
  • ✅ Integración con bases de datos mediante SQLAlchemy.
  • ✅ Autenticación y autorización de usuarios.
  • ✅ Proyecto práctico: API completa con Flask.
  • ✅ Cápsulas: buenas prácticas y seguridad en backend.
  • ✅ Co–creación: planificación e integración frontend–backend.
  • ✅ English Code: pruebas de código y blockchain.
  • ✅ Habilidades blandas: gestión del tiempo, trabajo en equipo y adaptabilidad.
