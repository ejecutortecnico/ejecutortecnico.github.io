Operadores en Python

En Python, los operadores se utilizan para realizar operaciones con variables y valores. A continuación, aprenderás sobre los operadores más comunes.

Operadores Aritméticos

Se usan para realizar operaciones matemáticas básicas.

  • +: Suma
  • -: Resta
  • *: Multiplicación
  • /: División
  • %: Módulo (resto)
  • **: Potencia
  • //: División entera
# Ejemplos de operadores aritméticos
x = 10
y = 3
print(x + y)   # 13
print(x - y)   # 7
print(x * y)   # 30
print(x / y)   # 3.333...
print(x % y)   # 1
print(x ** y)  # 1000
print(x // y)  # 3

Operadores de Comparación

Comparan dos valores y devuelven True o False.

  • ==: Igual a
  • !=: Diferente de
  • >: Mayor que
  • <: Menor que
  • >=: Mayor o igual a
  • <=: Menor o igual a
# Ejemplos de operadores de comparación
a = 5
b = 10
print(a == b)   # False
print(a != b)   # True
print(a > b)    # False
print(a < b)    # True
print(a >= b)   # False
print(a <= b)   # True

Operadores Lógicos

Se utilizan para combinar condiciones.

  • and: Devuelve True si ambas condiciones son verdaderas.
  • or: Devuelve True si al menos una condición es verdadera.
  • not: Devuelve el valor opuesto de la condición.
# Ejemplos de operadores lógicos
a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

Operadores de Asignación

Se utilizan para asignar valores a las variables.

  • =: Asignación simple
  • +=: Suma y asigna
  • -=: Resta y asigna
  • *=: Multiplica y asigna
  • /=: Divide y asigna
  • %=: Módulo y asigna
# Ejemplos de operadores de asignación
x = 5
x += 3   # x = x + 3 -> x es ahora 8
x -= 2   # x = x - 2 -> x es ahora 6
x *= 4   # x = x * 4 -> x es ahora 24
x /= 6   # x = x / 6 -> x es ahora 4.0

Conclusión

Los operadores en Python son esenciales para realizar cálculos, comparar valores y manejar la lógica del programa. Practicar estos operadores te ayudará a escribir código más eficiente y funcional.
