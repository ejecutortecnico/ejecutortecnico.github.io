Presentación de Matplotlib

¿Qué es Matplotlib?

Matplotlib es una de las bibliotecas más populares para la creación de gráficos en Python. Permite crear visualizaciones de datos en 2D con facilidad y flexibilidad.

Instalación

pip install matplotlib

Ejemplo básico

Uno de los gráficos más básicos que puedes crear con Matplotlib es una gráfica de líneas:

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [1, 4, 9, 16]

plt.plot(x, y)
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.title('Gráfico simple de Matplotlib')
plt.show()

Tipos de gráficos

Matplotlib soporta una gran variedad de tipos de gráficos:

  • Gráficos de líneas
  • Gráficos de barras
  • Histogramas
  • Gráficos de dispersión
  • Gráficos de pastel

Ejemplo de gráfico de barras

import matplotlib.pyplot as plt

categorias = ['A', 'B', 'C', 'D']
valores = [3, 7, 1, 8]

plt.bar(categorias, valores)
plt.xlabel('Categorías')
plt.ylabel('Valores')
plt.title('Gráfico de barras')
plt.show()

Personalización de gráficos

Matplotlib permite personalizar los gráficos agregando etiquetas, títulos, colores y estilos de línea.

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [1, 4, 9, 16]

plt.plot(x, y, color='green', linestyle='--', marker='o')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.title('Gráfico con estilo personalizado')
plt.grid(True)
plt.show()

Conclusión

Matplotlib es una herramienta poderosa para crear visualizaciones en Python. Su flexibilidad y facilidad de uso la convierten en una opción ideal para científicos de datos y desarrolladores.
