Preparación y Simulación (Lección 4): Herramientas y Bibliotecas en Python para IA

Exploraremos herramientas y bibliotecas fundamentales para la implementación de algoritmos de inteligencia artificial.

  • Familiarización con herramientas y bibliotecas de Python.
  • Configuración del entorno de desarrollo.
  • Preprocesamiento de datos y visualización.

1. Introducción a las Bibliotecas de Python para IA

NumPy: Manipulación de arreglos y operaciones matemáticas.

import numpy as np
array = np.array([1, 2, 3])
print(array * 2)

Pandas: Manipulación y análisis de datos.

import pandas as pd
data = {'name': ['Alice', 'Bob', 'Charlie'], 'age': [25, 30, 35]}
df = pd.DataFrame(data)
print(df)

Matplotlib: Visualización de datos.

import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])
plt.show()

2. Preprocesamiento de Datos

Limpieza de Datos:

  • Identificación y manejo de valores faltantes.
  • Detección y eliminación de duplicados.
df.dropna(inplace=True)
df.drop_duplicates(inplace=True)

Normalización y Estandarización:

from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
data_scaled = scaler.fit_transform(df[['age']])
print(data_scaled)

3. Implementación y Visualización

Implementación de un Pipeline Simple:

from sklearn.pipeline import Pipeline
from sklearn.linear_model import LinearRegression
pipeline = Pipeline([
    ('scaler', StandardScaler()),
    ('model', LinearRegression())
])
pipeline.fit(X, y)
predictions = pipeline.predict(X)
print(predictions)

Visualización de Resultados:

plt.scatter(X, y, color='blue')
plt.plot(X, predictions, color='red')
plt.show()
