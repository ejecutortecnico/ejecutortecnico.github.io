Bucle For en Python

Iterar sobre una lista:

frutas = ['manzana', 'banana', 'cereza']
for fruta en frutas:
    print(fruta)

Usando rangos en for

for i in range(5):
    print(i)

Información Adicional

El bucle for permite iterar sobre elementos en una lista o rango. Es útil para recorrer colecciones de datos de manera eficiente.

frutas = ['manzana', 'banana', 'cereza']
for fruta en frutas:
    print(fruta)  # Imprime cada fruta de la lista
