Tipos de Eventos en JavaScript

- Eventos de ratón (`click`, `dblclick`, `mouseover`, `mouseout`). - Eventos de teclado (`keydown`, `keyup`). - Eventos de formulario (`submit`, `change`).

Añadir y Eliminar Eventos

- Uso de `addEventListener` y `removeEventListener`: ```javascript let boton = document.getElementById("boton"); boton.addEventListener("click", function() { alert("Botón clicado!"); }); function mostrarMensaje() { alert("Mensaje mostrado!"); } boton.addEventListener("dblclick", mostrarMensaje); boton.removeEventListener("dblclick", mostrarMensaje); ```

Propagación de Eventos

.

- Conceptos de burbuja y captura de eventos: ```javascript let contenedor = document.getElementById("contenedor"); let elemento = document.getElementById("elemento"); contenedor.addEventListener("click", function() { alert("Contenedor clicado!"); }, true); // Captura elemento.addEventListener("click", function() { alert("Elemento clicado!"); }); // Burbuja
