Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje utilizado para estilizar páginas web, permitiendo controlar el aspecto visual de elementos HTML.

1. Cómo Vincular CSS a HTML

Existen tres maneras de aplicar CSS a una página HTML:

  • En línea: Usando el atributo style en elementos HTML.
  • Interno: Incluyendo CSS dentro de una etiqueta <style> en el <head>.
  • Externo: Usando un archivo CSS externo vinculado con <link>.

2. Selectores CSS

Los selectores permiten elegir qué elementos HTML estilizar. Los principales selectores son:

  • Selector de elemento: Selecciona todas las etiquetas de un tipo (e.g., p).
  • Selector de clase: Selecciona elementos con una clase específica (e.g., .clase).
  • Selector de ID: Selecciona un elemento con un ID único (e.g., #id).
/* Selector de elemento */
p {
    color: blue;
}

/* Selector de clase */
.cuadro {
    background-color: lightgray;
}

/* Selector de ID */
#titulo {
    font-size: 24px;
}

3. Colores y Fondos

CSS permite agregar colores y fondos a los elementos. Puedes especificar colores en palabras clave, hexadecimal, RGB o HSL.

/* Color de texto */
p {
    color: #333333;
}

/* Fondo de un elemento */
body {
    background-color: #f4f4f4;
}

/* Fondo con imagen */
body {
    background-image: url("imagen.png");
}

4. Propiedades de Texto

CSS tiene propiedades para controlar el estilo, tamaño, alineación y otras características del texto.

  • font-size: Tamaño del texto
  • color: Color del texto
  • text-align: Alineación del texto
  • font-weight: Grosor de la fuente
/* Propiedades de texto */
h1 {
    font-size: 32px;
    color: #ff00c3;
    text-align: center;
    font-weight: bold;
}

5. Margen y Relleno

El margen (margin) controla el espacio exterior, mientras que el relleno (padding) controla el espacio interior de un elemento.

/* Ejemplo de margen y relleno */
.container {
    margin: 20px;
    padding: 10px;
    border: 1px solid #ddd;
}

6. Bordes y Sombras

CSS permite personalizar los bordes y agregar sombras a los elementos.

/* Ejemplo de borde */
.caja {
    border: 2px solid #333;
    border-radius: 10px;
}

/* Ejemplo de sombra */
.caja {
    box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.3);
}

7. Modelo de Caja

El modelo de caja describe cómo se disponen los elementos en una página y cómo se calculan sus dimensiones.

/* Modelo de caja */
.caja {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 1px solid #333;
    margin: 10px;
}

Conclusión

CSS es fundamental para darle estilo a una página web. Practica estos conceptos para mejorar tus habilidades en diseño y estilo web.
