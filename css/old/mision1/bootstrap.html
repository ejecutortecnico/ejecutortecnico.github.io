Bootstrap Básico

Introducción a Bootstrap

Bootstrap es un framework CSS de código abierto que facilita el desarrollo de sitios web modernos, responsivos y visualmente atractivos. Contiene herramientas como un sistema de grid, componentes UI, y clases de utilidades.

1. Instalación de Bootstrap

La forma más rápida de incluir Bootstrap es a través de un CDN. Esto asegura que los usuarios carguen el framework desde servidores rápidos y confiables.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

2. Sistema de Grid

El sistema de grid de Bootstrap es fundamental para crear diseños responsivos. Se basa en un contenedor (container), filas (row) y columnas (col).

<div class="container">
    <div class="row">
        <div class="col-md-6">Columna 1 (6/12)</div>
        <div class="col-md-6">Columna 2 (6/12)</div>
    </div>
</div>

Nota: Bootstrap usa una grilla de 12 columnas, donde col-md-6 ocupa la mitad del ancho en dispositivos medianos.

3. Componentes Básicos

Bootstrap incluye componentes preconstruidos para interfaces, como botones, alertas, y tarjetas:

<button class="btn btn-primary">Botón Primario</button>
<div class="alert alert-warning">¡Cuidado! Esta es una alerta de advertencia.</div>
<div class="card" style="width: 18rem;">
    <img src="imagen.jpg" class="card-img-top" alt="Imagen">
    <div class="card-body">
        <h5 class="card-title">Título de la Tarjeta</h5>
        <p class="card-text">Texto de la tarjeta con información descriptiva.</p>
    </div>
</div>

4. Clases de Utilidad

Bootstrap ofrece clases de utilidad para personalizar el espaciado, el tamaño del texto, el color y mucho más.

<div class="p-3 mb-2 bg-info text-white">
    Este div tiene un fondo azul claro y texto blanco, con un padding de 3 y un margen de 2.
</div>

Ejemplos de clases de utilidad:

  • m-3: Margen de 3
  • p-2: Padding de 2
  • text-center: Texto centrado
  • bg-success: Fondo verde

5. Ejemplo Práctico

Combina varios elementos de Bootstrap para crear una página simple y responsiva.

<!DOCTYPE html>
<html lang="es">
<head>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container">
        <nav class="navbar navbar-expand-lg navbar-light bg-light">
            <a class="navbar-brand" href="#">Mi Sitio</a>
            <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav">
                    <li class="nav-item"><a class="nav-link" href="#">Inicio</a></li>
                    <li class="nav-item"><a class="nav-link" href="#">Servicios</a></li>
                </ul>
            </div>
        </nav>

        <div class="row mt-4">
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Bienvenidos</h5>
                        <p class="card-text">Explora nuestro sitio web con Bootstrap.</p>
                        <a href="#" class="btn btn-primary">Más Información</a>
                    </div>
                </div>
            </div>
            <div class="col-md-6">
                <div class="alert alert-info">Este sitio está diseñado con Bootstrap.</div>
            </div>
        </div>
    </div>
</body>
</html>

Conclusión

Bootstrap simplifica el desarrollo web y permite crear interfaces modernas de manera rápida. Con su sistema de grid, componentes y clases de utilidad, es posible crear diseños responsivos y atractivos.
