Tipos de Datos en JavaScript

JavaScript tiene varios tipos de datos fundamentales que se utilizan para almacenar y manipular información.

  • String: Cadenas de texto.
  • Number: Números, tanto enteros como decimales.
  • Boolean: Valores lógicos, verdadero (true) o falso (false).
  • Undefined: Una variable declarada pero sin valor asignado.
  • Null: Intencionalmente vacío o sin valor.
  • Object: Estructura compleja que puede contener pares clave-valor.
  • Symbol: Tipo de dato único y primitivo introducido en ES6.
