Variables en Python

Las variables son contenedores para almacenar datos. En Python, no es necesario declarar el tipo de dato, ya que es de tipado dinámico.

Declaración de Variables

En Python, asignar una variable es sencillo. Solo necesitas un nombre y el valor que deseas almacenar.

# Declaración de variables
nombre = "Ana"
edad = 25
altura = 1.75

Tipos de Datos Comunes

Python soporta varios tipos de datos básicos:

  • str: Cadenas de texto.
  • int: Números enteros.
  • float: Números decimales.
  • bool: Booleanos (True/False).
# Ejemplos de tipos de datos
texto = "Hola"
numero_entero = 10
numero_decimal = 10.5
es_mayor = True

Cambio de Valores

Las variables en Python pueden cambiar de valor durante la ejecución del programa.

# Asignar un nuevo valor a una variable
edad = 30
print(edad)  # Imprime: 30

Buenas Prácticas

Para escribir código claro y fácil de entender, sigue estas buenas prácticas:

  • Usa nombres descriptivos para tus variables.
  • Usa minúsculas y guiones bajos para separar palabras (nombre_completo).
  • Evita palabras reservadas como nombres de variables.
# Ejemplo de buena práctica
nombre_completo = "Ana García"
edad_usuario = 28

Conclusión

Las variables son esenciales en cualquier lenguaje de programación. Python facilita su uso al no requerir declarar el tipo de dato explícitamente, lo que hace el código más simple y legible.
