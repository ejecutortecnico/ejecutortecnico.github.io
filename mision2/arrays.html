Arrays en JavaScript

Aprende a manejar colecciones de datos con Arrays en JavaScript.

1. Creación de Arrays

Cómo crear un array en JavaScript:

// Crear un array vacío
let miArray = [];

// Crear un array con elementos
let numeros = [1, 2, 3, 4, 5];

2. Métodos Básicos de Arrays

Agregar y eliminar elementos:

// Agregar un elemento al final
numeros.push(6); // [1, 2, 3, 4, 5, 6]

// Eliminar el último elemento
numeros.pop(); // [1, 2, 3, 4, 5]

3. Iterar sobre Arrays

Usando un bucle para recorrer un array:

let frutas = ['manzana', 'banana', 'naranja'];
for(let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

4. Otros Métodos Útiles

Algunos métodos adicionales:

// Encontrar elementos
let indice = numeros.indexOf(3); // devuelve 2

// Combinar arrays
let otrosNumeros = [6, 7];
let combinado = numeros.concat(otrosNumeros); // [1, 2, 3, 4, 5, 6, 7]
