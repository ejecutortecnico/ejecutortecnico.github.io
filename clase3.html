Etiqueta DIV

El elemento de HTML div es el contenedor genérico para el contenido. No tiene efecto en el contenido o en el diseño hasta que se le aplica un estilo de alguna manera usando CSS

Modelo de Caja

El modelo de caja comprende algunas propiedades que definen como se verá tu sitio: box-sizing, border, padding y margin. Estas propiedades finalmente afectan como tu contenido será desplegado.

Display FLex

El módulo de diseño de caja flexible facilita el diseño de una estructura de diseño flexible y responsiva sin utilizar flotación ni posicionamiento.
Para comenzar a utilizar el modelo Flexbox, primero debemos definir un contenedor flexible.

Display Grid

El módulo de diseño de cuadrícula CSS ofrece un sistema de diseño basado en cuadrícula, con filas y columnas, lo que facilita el diseño de páginas web sin tener que utilizar flotantes y posicionamiento.
