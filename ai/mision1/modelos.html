¿Qué significa “aprender” para una máquina?

Para una máquina, aprender NO significa pensar como un humano.

Aprender significa ajustar su comportamiento usando datos.

  • Recibe datos
  • Detecta patrones
  • Ajusta valores internos
  • Mejora sus resultados con el tiempo

¿Cómo aprende una máquina?

  • No tiene conciencia ni emociones
  • No entiende, reconoce patrones
  • Aprende usando matemáticas y estadísticas

Ejemplo:

Un filtro de spam aprende analizando miles de correos y reduce errores con la experiencia.

¿Qué es un modelo de Inteligencia Artificial?

Un modelo de IA es un sistema que aprende a realizar tareas específicas a partir de datos. Utiliza algoritmos para identificar patrones y tomar decisiones sin estar programado explícitamente para cada tarea.

Modelos de IA: Tipos Generales

  • Modelos supervisados (clasificación, regresión)
  • Modelos no supervisados (agrupamiento, reducción de dimensionalidad)
  • Aprendizaje por refuerzo

¿Qué son las tareas en Machine Learning?

Las tareas principales del aprendizaje automático se agrupan en:

  • Clasificación
  • Regresión
  • Clustering (agrupamiento)
  • Reducción de dimensionalidad

Clasificación

Predice una categoría o clase a la que pertenece una observación.

Ejemplo: Clasificar emails como spam o no spam.

from sklearn.linear_model import LogisticRegression
              modelo = LogisticRegression().fit(X_train, y_train)
              y_pred = modelo.predict(X_test)

Regresión

Predice un valor numérico continuo a partir de las variables de entrada.

Ejemplo: Estimar el precio de una casa según su tamaño, ubicación, etc.

from sklearn.linear_model import LinearRegression
              modelo = LinearRegression().fit(X_train, y_train)
              precio = modelo.predict([[100, 3, 2]])

Clustering (Agrupamiento)

Encuentra grupos naturales en los datos sin usar etiquetas.

Ejemplo: Agrupar clientes con hábitos similares de compra.

from sklearn.cluster import KMeans
              modelo = KMeans(n_clusters=3).fit(X)
              grupo = modelo.predict([[20, 500]])

Reducción de Dimensionalidad

Reduce la cantidad de variables conservando la mayor información posible.

Ejemplo: Visualizar datos de alta dimensión en 2D con PCA.

from sklearn.decomposition import PCA
              pca = PCA(n_components=2).fit_transform(X)

Modelos de IA en Scikit-Learn

Scikit-learn es una biblioteca de Python para aprendizaje automático que ofrece una amplia variedad de modelos para clasificación, regresión, clustering y reducción de dimensionalidad.

Clasificación vs Regresión

  • Clasificación: predice categorías. Ej: spam/no spam
  • Regresión: predice valores numéricos. Ej: precio de una casa

Árbol de Decisión (DecisionTreeClassifier)

Modelo interpretable basado en reglas. Ideal para tareas con decisiones jerárquicas.

from sklearn.tree import DecisionTreeClassifier

Random Forest (RandomForestClassifier)

Ensamble de muchos árboles de decisión. Mejora la precisión y reduce el sobreajuste.

from sklearn.ensemble import RandomForestClassifier

k-Nearest Neighbors (KNeighborsClassifier)

Clasifica según las clases de los vecinos más cercanos. Simple y efectivo.

from sklearn.neighbors import KNeighborsClassifier

Support Vector Machine (SVC)

Encuentra el hiperplano que mejor separa las clases. Eficiente en espacios de alta dimensión.

from sklearn.svm import SVC

Regresión Logística (LogisticRegression)

Modelo lineal para clasificación binaria. Muy usado por su simplicidad y rendimiento.

from sklearn.linear_model import LogisticRegression

Regresión Lineal (LinearRegression)

Predice valores numéricos como combinación lineal de las variables de entrada.

from sklearn.linear_model import LinearRegression

Árbol de Decisión para Regresión (DecisionTreeRegressor)

Como el clasificador, pero para predecir valores continuos.

from sklearn.tree import DecisionTreeRegressor

Random Forest Regressor

Conjunto de árboles de decisión para tareas de regresión.

from sklearn.ensemble import RandomForestRegressor

Clustering con K-Means

Modelo no supervisado para agrupar datos similares en K grupos.

from sklearn.cluster import KMeans

Reducción de Dimensionalidad: PCA

PCA (Análisis de Componentes Principales) reduce dimensiones manteniendo la varianza.

from sklearn.decomposition import PCA

Comparación de Modelos en Scikit-Learn

Scikit-learn ofrece una interfaz común para entrenar y evaluar todos estos modelos:


              modelo.fit(X_train, y_train)
              predicciones = modelo.predict(X_test)
              accuracy_score(y_test, predicciones)

Redes Neuronales Artificiales

Inspiradas en el cerebro humano, las redes neuronales están compuestas por capas de nodos (neuronas) que transforman datos de entrada en salida a través de funciones matemáticas.

input → capa_oculta → output

Estructura de una Red Neuronal

  • Capa de entrada: recibe los datos
  • Capas ocultas: procesan la información
  • Capa de salida: produce el resultado

¿Por qué usar redes neuronales?

  • Pueden modelar relaciones no lineales complejas
  • Son la base de muchas tecnologías modernas como reconocimiento de voz, visión por computadora y chatbots
