Strings en JavaScript

Aprende cómo trabajar con cadenas de texto en JavaScript.

1. Creación de Strings

Existen diferentes maneras de crear cadenas:

// Usando comillas simples
let saludo = 'Hola, mundo';

// Usando comillas dobles
let frase = "JavaScript es genial";

// Usando plantillas literales (template literals)
let nombre = 'Ana';
let mensaje = `Hola, ${nombre}`;

2. Propiedades de los Strings

Los strings tienen propiedades útiles, como length:

let texto = "JavaScript";
console.log(texto.length); // 10

3. Métodos Comunes de Strings

Algunos métodos útiles para trabajar con strings:

// Convertir a mayúsculas
let texto = "javascript";
console.log(texto.toUpperCase()); // "JAVASCRIPT"

// Reemplazar parte de un string
let saludo = "Hola, mundo";
let nuevoSaludo = saludo.replace("mundo", "amigo");
console.log(nuevoSaludo); // "Hola, amigo"

4. Plantillas Literales

Las plantillas literales permiten incluir variables de manera sencilla:

let producto = 'Laptop';
let precio = 1500;
let mensaje = `El precio de la ${producto} es $${precio}`;
console.log(mensaje);
