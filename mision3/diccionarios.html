Ejercicios Prácticos de Diccionarios en Python

Los diccionarios en Python son colecciones desordenadas de pares clave-valor. A continuación, te presentamos algunos ejercicios prácticos para trabajar con ellos.

Ejercicio 1: Crear un Diccionario

Crea un diccionario llamado alumno que contenga las claves: "nombre", "edad", "curso". Luego, imprime el diccionario completo.

# Solución
alumno = {
    "nombre": "Juan",
    "edad": 20,
    "curso": "Matemáticas"
}

# Imprimir el diccionario
print(alumno)

Ejercicio 2: Acceder a Elementos

Dado el diccionario alumno, imprime solo el valor de la clave "nombre".

# Solución
print(alumno["nombre"])  # Imprime: Juan

Ejercicio 3: Agregar un Elemento

Agrega al diccionario alumno la clave "nota" con el valor 8.5.

# Solución
alumno["nota"] = 8.5
print(alumno)

Ejercicio 4: Eliminar un Elemento

Elimina la clave "curso" del diccionario alumno.

# Solución
del alumno["curso"]
print(alumno)

Ejercicio 5: Recorrer un Diccionario

Utiliza un bucle for para recorrer e imprimir las claves y los valores del diccionario alumno.

# Solución
for clave, valor in alumno.items():
    print(clave, ":", valor)

Ejercicio 6: Diccionario Anidado

Crea un diccionario llamado clase que contenga dos alumnos, cada uno representado como un diccionario con su "nombre", "edad" y "nota".

# Solución
clase = {
    "alumno1": {"nombre": "Ana", "edad": 21, "nota": 9.0},
    "alumno2": {"nombre": "Carlos", "edad": 22, "nota": 7.5}
}

# Imprimir el diccionario completo
print(clase)

Ejercicio 7: Modificar un Valor en Diccionario Anidado

Cambia la nota del "alumno1" a 9.5 en el diccionario clase.

# Solución
clase["alumno1"]["nota"] = 9.5
print(clase["alumno1"])

Conclusión

Trabajar con diccionarios en Python te permite gestionar datos de manera eficiente. Practicar con diferentes escenarios te ayudará a dominar esta estructura.
