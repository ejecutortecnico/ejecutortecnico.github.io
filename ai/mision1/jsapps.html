¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web. Define la estructura básica de una página web utilizando etiquetas como <div>, <h1>, <p>, entre otras.

Estructura Básica de HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Página Web</title>
  </head>
  <body>
    <h1>Bienvenido a mi página</h1>
    <p>Este es un ejemplo de una página web sencilla</p>
  </body>
</html>

Este es un ejemplo básico de una página web en HTML, con etiquetas para encabezados y párrafos.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para definir el estilo de una página web, como los colores, las fuentes, y la disposición de los elementos en la pantalla.

Integrando CSS con HTML

<head>
  <style>
    body {
      background-color: lightblue;
      font-family: Arial, sans-serif;
    }
    h1 {
      color: darkblue;
    }
  </style>
</head>

Este fragmento de código muestra cómo puedes incluir estilos CSS directamente en un archivo HTML dentro de la etiqueta <head>.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web, como animaciones, formularios dinámicos y respuestas a eventos de usuario.

Integrando JavaScript con HTML

<body>
  <button onclick="alert('¡Hola, Mundo!')">Haz clic aquí</button>
</body>

Este ejemplo muestra cómo se puede integrar JavaScript dentro de una página HTML. En este caso, cuando el usuario hace clic en el botón, aparece un mensaje emergente.

Conexión entre HTML, CSS y JavaScript

En una página web, HTML define la estructura, CSS le da estilo, y JavaScript agrega interactividad. Juntos crean una experiencia web dinámica y atractiva.

Ejemplo Completo de HTML, CSS y JavaScript

<html>
  <head>
    <title>Ejemplo Web</title>
    <style>
      body {
        font-family: Arial, sans-serif;
        background-color: lightgray;
      }
      h1 {
        color: #333;
      }
    </style>
  </head>
  <body>
    <h1>Hola, Mundo!</h1>
    <button onclick="alert('¡Has hecho clic en el botón!')">Haz clic aquí</button>
  </body>
</html>

Este es un ejemplo completo de cómo HTML, CSS y JavaScript se pueden integrar en una página web simple.
