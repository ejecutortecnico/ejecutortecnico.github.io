Manejo de Datos con Pandas y Matplotlib

Python ofrece poderosas bibliotecas como Pandas y Matplotlib para la manipulación y visualización de datos. Esta presentación explora cómo trabajar con datos de manera eficiente.

Introducción a Pandas

Pandas es una biblioteca para la manipulación y el análisis de datos en Python. Se utiliza para trabajar con estructuras de datos como Series y DataFrames.

# Instalar Pandas
!pip install pandas

# Importar Pandas
import pandas as pd

Cargando Datos con Pandas

Pandas facilita la carga de datos desde diferentes fuentes, como archivos CSV, Excel, y bases de datos SQL. Ejemplo de carga de un archivo CSV:

# Cargar un archivo CSV en un DataFrame
df = pd.read_csv("data.csv")

# Ver las primeras filas del DataFrame
print(df.head())

Manipulación de Datos con Pandas

Pandas permite manipular los datos de manera sencilla, incluyendo selección de columnas, filtrado de datos, y agregación.

# Seleccionar una columna
columna = df["Nombre"]

# Filtrar datos
filtro = df[df["Edad"] > 30]

# Agregar datos (promedio de una columna)
promedio = df["Salario"].mean()

Visualización de Datos con Matplotlib

Matplotlib es una biblioteca para la creación de gráficos y visualización de datos en Python.

# Instalar Matplotlib
!pip install matplotlib

# Importar Matplotlib
import matplotlib.pyplot as plt

Gráficos Básicos con Matplotlib

Matplotlib permite crear gráficos simples como gráficos de líneas, barras y dispersión. Ejemplo de un gráfico de líneas:

# Crear un gráfico de líneas
plt.plot(df["Fecha"], df["Valor"])
plt.title("Gráfico de Líneas")
plt.xlabel("Fecha")
plt.ylabel("Valor")
plt.show()

Gráficos de Barras con Matplotlib

También podemos crear gráficos de barras para comparar diferentes categorías de datos.

# Crear un gráfico de barras
plt.bar(df["Categoría"], df["Cantidad"])
plt.title("Gráfico de Barras")
plt.xlabel("Categoría")
plt.ylabel("Cantidad")
plt.show()

Combinar Pandas y Matplotlib

Es común utilizar Pandas para la manipulación de datos y luego Matplotlib para la visualización. Ejemplo de combinación de ambas bibliotecas:

# Agrupar datos y graficar
df_agrupado = df.groupby("Categoría")["Cantidad"].sum()
df_agrupado.plot(kind="bar", title="Cantidad por Categoría")
plt.show()

Conclusión

Pandas y Matplotlib son herramientas esenciales para el manejo y visualización de datos en Python. Con ellas, puedes realizar análisis de datos y crear visualizaciones informativas de manera eficiente.
