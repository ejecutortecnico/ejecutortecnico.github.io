¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura del contenido usando etiquetas.

Estructura básica de un documento HTML

<!DOCTYPE html>
<html>
<head>
    <title>Título de la página</title>
</head>
<body>
    <h1>Encabezado</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

Etiquetas HTML más comunes

  • <h1> a <h6>: Encabezados
  • <p>: Párrafos
  • <a>: Enlaces
  • <img>: Imágenes
  • <ul> y <ol>: Listas
  • <div> y <span>: Contenedores

Atributos en HTML

Los atributos proporcionan información adicional sobre los elementos HTML. Se especifican dentro de la etiqueta de apertura.

<a href="https://www.example.com">Visita mi sitio</a>
<img src="imagen.jpg" alt="Descripción de la imagen">

Imágenes en HTML

<img src="imagen.jpg" alt="Descripción de la imagen">

src especifica la ruta de la imagen y alt proporciona un texto alternativo.

Enlaces en HTML

<a href="https://www.example.com">Haz clic aquí</a>

La etiqueta <a> se utiliza para crear enlaces. El atributo href especifica la URL.

Listas en HTML

Existen dos tipos de listas en HTML:

  • <ul>: Lista desordenada (con viñetas)
  • <ol>: Lista ordenada (con números)
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

<ol>
  <li>Primero</li>
  <li>Segundo</li>
</ol>

Formularios en HTML

Los formularios permiten a los usuarios enviar datos a un servidor.

<form action="/submit" method="POST">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
  <input type="submit" value="Enviar">
</form>

Conclusión

HTML es el lenguaje base para la creación de sitios web. Entender su estructura y etiquetas es fundamental para desarrollar páginas web efectivas y accesibles.
