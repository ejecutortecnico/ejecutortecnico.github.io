Pandas en Python

Introducción a Pandas

Pandas es una biblioteca esencial para la manipulación y análisis de datos en Python.

1. Series en Pandas

Las Series son estructuras de datos unidimensionales en Pandas.

import pandas as pd

# Crear una Serie
mi_serie = pd.Series([1, 2, 3, 4, 5])
print(mi_serie)

2. DataFrames en Pandas

Los DataFrames son estructuras de datos bidimensionales con etiquetas en filas y columnas.

# Crear un DataFrame
data = {'Nombre': ['Ana', 'Luis', 'Carlos'], 'Edad': [23, 34, 29]}
df = pd.DataFrame(data)
print(df)

3. Operaciones Básicas

Pandas ofrece funciones rápidas para analizar datos.

# Seleccionar columna
print(df['Nombre'])

# Descripción estadística
print(df.describe())

4. Filtrado de Datos

Filtrar datos en un DataFrame basado en condiciones específicas.

# Filtrar personas mayores de 30 años
filtro = df[df['Edad'] > 30]
print(filtro)

5. Manipulación de Datos

Agregar, eliminar o modificar datos en un DataFrame.

# Añadir una columna
df['Ciudad'] = ['Madrid', 'Barcelona', 'Valencia']

# Eliminar una columna
df = df.drop(columns=['Edad'])

print(df)

Conclusión

Pandas es una herramienta poderosa para análisis de datos, permitiendo transformar y limpiar datasets rápidamente.
