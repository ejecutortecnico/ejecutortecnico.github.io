Lección 02: Herramientas Básicas para el Análisis de Datos

1. Introducción a Python: Visión General

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, conocido por su sintaxis simple y legible[cite: 358]. Creado por Guido van Rossum en 1991, es muy popular en ciencia de datos, desarrollo web e inteligencia artificial[cite: 359].

Características Clave:

  • Simplicidad y Legibilidad[cite: 360].
  • Versatilidad (no solo análisis de datos)[cite: 362, 363].
  • Extensa biblioteca de herramientas y paquetes (pandas, numpy, matplotlib, scikit-learn, etc.)[cite: 365, 366, 367, 368, 369, 370].
  • Interactividad (Jupyter Notebooks)[cite: 372].
  • Comunidad activa y soporte[cite: 374].
  • Portabilidad y Flexibilidad (multiplataforma)[cite: 376].
  • Integración con otras tecnologías (C, C++, SQL)[cite: 378, 379].
  • Automatización de tareas repetitivas[cite: 381].
  • Líder en Aprendizaje Automático e IA[cite: 383].

2. Python: Ejemplo Práctico de Código

Ejemplo de cómo calcular la media de una lista en Python:


# Calcular la media de una lista de números en Python
numeros = [10, 20, 30, 40, 50]
media = sum(numeros) / len(numeros)
print(f"La media es: {media}")
# Salida: La media es: 30.0

Este código demuestra la simplicidad de Python para realizar cálculos básicos[cite: 362].

3. Introducción a R: Visión General

R es un lenguaje y entorno de software libre diseñado específicamente para análisis estadístico y visualización de datos[cite: 389]. Desarrollado por Ross Ihaka y Robert Gentleman, es muy popular entre estadísticos y científicos de datos[cite: 390].

Características Clave:

  • Especialización en Análisis Estadístico (pruebas, modelado lineal/no lineal, series temporales, etc.)[cite: 391].
  • Extensa colección de paquetes estadísticos (CRAN)[cite: 392].
  • Visualización de Datos Avanzada (gráficos de alta calidad, paquete ggplot2)[cite: 394, 395].
  • Amplia colección de paquetes para diversas disciplinas (+18,000 en CRAN)[cite: 396].
  • Capacidades de Manipulación de Datos (dplyr, tidyr)[cite: 398].
  • Interactividad y Creación de Aplicaciones Web (Shiny)[cite: 399].
  • Entorno de Desarrollo Integrado (RStudio)[cite: 402].
  • Soporte para Grandes Conjuntos de Datos (data.table, sparklyr)[cite: 407, 408].
  • Comunidad Activa y Recursos Abundantes[cite: 409].
  • Portabilidad y Compatibilidad (multiplataforma, integración con otros lenguajes)[cite: 412, 413].

4. R: Ejemplo Práctico de Código

Ejemplo de visualización básica con ggplot2 en R:


# Ejemplo de visualización con ggplot2 en R
library(ggplot2)

# Utiliza el dataset 'mpg' que viene con ggplot2
ggplot(mpg, aes(x=displ, y=hwy, color=class)) +
  geom_point() +
  labs(title="Consumo de Combustible en Automóviles",
       x="Desplazamiento del Motor (L)",
       y="Millas por Galón en Carretera") +
  theme_minimal()

Este código genera un gráfico de dispersión mostrando la relación entre el desplazamiento del motor y el consumo de combustible, coloreado por clase de vehículo[cite: 396].

7. Instalación de Python y Entornos Virtuales

Instalación de Python:

  • Descargar desde python.org[cite: 497].
  • En Windows, marcar "Add Python to PATH"[cite: 500].
  • Verificar con `python --version` o `python3 --version`[cite: 501, 502, 504].

Entornos Virtuales (con `virtualenv` o `venv`):

  • Crean instalaciones aisladas de Python para proyectos[cite: 505].
  • Evitan conflictos de dependencias[cite: 506].
  • Instalar: `pip install virtualenv` (si se usa virtualenv)[cite: 508].
  • Crear: `python -m venv nombre_entorno`[cite: 1177].
  • Activar: `source nombre_entorno/bin/activate` (Linux/macOS) o `.\nombre_entorno\Scripts\activate` (Windows)[cite: 1177].
  • Instalar paquetes dentro del entorno activado: `pip install nombre_paquete`[cite: 510].
  • Gestionar dependencias con `requirements.txt` (`pip freeze > requirements.txt`, `pip install -r requirements.txt`)[cite: 510].

8. Instalación de R y RStudio

Instalación de R:

  • Descargar desde cran.r-project.org[cite: 512].
  • Seguir instrucciones del instalador para el SO específico (Windows .exe, macOS .pkg, Linux gestor de paquetes)[cite: 513, 514, 516, 518].
  • Verificar abriendo la consola R o ejecutando `R` en la terminal[cite: 515, 517, 519].

Instalación de RStudio:

  • Descargar desde rstudio.com/products/rstudio/download/[cite: 521].
  • Seleccionar la versión gratuita "RStudio Desktop"[cite: 521].
  • Seguir instrucciones del instalador (Windows .exe, macOS .dmg, Linux .deb/.rpm)[cite: 522, 523, 524, 525].
  • Verificar abriendo RStudio[cite: 525]. RStudio detectará la instalación de R[cite: 526].

10. Uso de Bibliotecas: pandas y numpy (Python)

pandas:

  • Estructuras: DataFrame (tabla 2D) y Series (columna 1D)[cite: 542, 545].
  • Carga datos: `pd.read_csv()`, `pd.read_excel()`, etc[cite: 549].
  • Exploración: `df.head()`, `df.info()`, `df.describe()`[cite: 1185].
  • Manipulación: Filtrado (`df[condicion]`), selección (`df[['col']]`), nuevas columnas (`df['nueva'] = ...`), agrupar (`df.groupby()`)[cite: 550, 551, 553].
  • Manejo de nulos: `df.isnull().sum()`, `df.dropna()`, `df.fillna()`[cite: 555].

numpy:

  • Estructura: `ndarray` (array N-dimensional)[cite: 562].
  • Operaciones vectorizadas rápidas (elemento a elemento)[cite: 1190].
  • Funciones matemáticas: `np.mean()`, `np.std()`, `np.sqrt()`, etc[cite: 1192, 568].
  • Álgebra lineal: `np.dot()` (producto de matrices), `np.linalg.inv()` (inversa)[cite: 569, 577].
  • Manipulación: Indexación, slicing, `reshape()`, `transpose()`[cite: 570, 572].

# Ejemplo pandas y numpy
import pandas as pd
import numpy as np

data = {'col1': [1, 2, 3, 4], 'col2': [10, 20, 30, 40]}
df = pd.DataFrame(data)

# Calcular la media de col1 usando numpy
media_np = np.mean(df['col1'].values)
print(f"Media calculada con NumPy: {media_np}")

# Añadir columna usando pandas
df['col3'] = df['col1'] * df['col2']
print("DataFrame con nueva columna:")
print(df)

11. Uso de Bibliotecas: matplotlib y tidyverse (R)

matplotlib (Python, base para seaborn):

  • Gráficos básicos: `plt.plot()` (líneas), `plt.bar()` (barras), `plt.scatter()` (dispersión), `plt.hist()` (histograma)[cite: 581, 582, 583, 585].
  • Personalización: `plt.title()`, `plt.xlabel()`, `plt.ylabel()`, `plt.legend()`, `plt.style.use()`[cite: 587, 589].
  • Subplots: `plt.subplot()` para múltiples gráficos[cite: 590].
  • Guardar: `plt.savefig()`[cite: 591].

tidyverse (R, incluye ggplot2 y dplyr):

  • dplyr: Manipulación de datos. Verbos clave: `filter()`, `select()`, `mutate()`, `group_by()`, `summarise()`, `arrange()`[cite: 1301, 1302, 1303, 1304, 1305, 1306]. Usa el operador `%>%` (pipe) para encadenar operaciones[cite: 598].
  • ggplot2: Visualización basada en "gramática de gráficos". Estructura: `ggplot(data, aes(x=..., y=...)) + geom_...()`[cite: 1309]. `geom_bar()`, `geom_histogram()`, `geom_point()`, `geom_line()`, etc[cite: 1310, 1311, 1313, 1314]. Personalización con `labs()`, `ggtitle()`, `theme_...()`[cite: 1315, 1404].

# Ejemplo tidyverse en R
library(dplyr)
library(ggplot2)

# Crear un dataframe de ejemplo
data <- data.frame(
  categoria = sample(c("A", "B", "C"), 100, replace = TRUE),
  valor = rnorm(100, 50, 10)
)

# Filtrar y resumir datos con dplyr
resumen <- data %>%
  filter(valor > 40) %>%
  group_by(categoria) %>%
  summarise(media_valor = mean(valor, na.rm = TRUE))

print(resumen)

# Crear un gráfico de barras con ggplot2
ggplot(data, aes(x = categoria, y = valor, fill = categoria)) +
  geom_boxplot() + # O geom_violin(), geom_bar(stat="identity") si ya resumido
  labs(title = "Distribución de Valor por Categoría",
       x = "Categoría",
       y = "Valor") +
  theme_minimal()
