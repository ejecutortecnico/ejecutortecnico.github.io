Exploraremos decoradores, generadores, manejo avanzado de excepciones y manejo de paquetes en Python.

1. Decoradores

Un decorador es una función que modifica el comportamiento de otra función.

def decorador(func):
    def envoltura():
        print("Inicio del decorador")
        func()
        print("Fin del decorador")
    return envoltura

@decorador
def saludo():
    print("Hola")

saludo()

2. Generadores

Los generadores permiten crear iteradores de una manera más eficiente usando yield.

def contador(max):
    n = 0
    while n < max:
        yield n
        n += 1

for num in contador(5):
    print(num)  # Output: 0 1 2 3 4

3. Manejo Avanzado de Excepciones

Python permite manejar múltiples tipos de excepciones y finalmente bloques con finally.

try:
    resultado = 10 / 0
except ZeroDivisionError:
    print("Error: División por cero")
except Exception as e:
    print(f"Error inesperado: {e}")
finally:
    print("Finalizando el bloque")

4. Paquetes en Python

Los paquetes permiten organizar módulos en carpetas.

# Estructura de un paquete
mi_paquete/
    __init__.py
    modulo1.py
    modulo2.py

# Importar un módulo del paquete
from mi_paquete import modulo1
